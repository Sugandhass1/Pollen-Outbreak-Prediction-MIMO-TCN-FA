# Pollen-Outbreak-Prediction-MIMO-TCN-FA
This repository contains datasets, preprocessing files, and supporting materials for the research work involving Advanced Framework for Predicting Pollen Outbreaks Using Multi-Input Multi-Output Temporal Convolutional Networks and Firefly Algorithm.
The framework aims to improve prediction accuracy by leveraging temporal weather features and evolutionary hyperparameter tuning.

Repository Contents
Weather Data Extraction – Raw and processed meteorological datasets
Japan vs India Weather – Comparative weather datasets used for analysis
Documentation – Supporting files related to experiments and preprocessing
